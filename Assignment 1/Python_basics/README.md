# Assignment 1

This folder contains five simple Python programs written in a beginner-friendly style:

1. `primenumber_check.py` - checks whether a number is prime.
2. `palindrome_check.py` - checks whether text is a palindrome.
3. `simple_even_odd.py` - checks whether a number is even or odd.
4. `matrix_product.py` - multiplies two matrices.
5. `factorial.py` - calculates factorial of a non-negative number.
